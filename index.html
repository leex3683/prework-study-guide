Git

  • git clone git@github.com:leex3683/prework-study-guide.git
  • git remote -v allows fetching/pushing
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • (git checkout _____ ) moves branches
  • first change code, and save code
  • then stage with "git add -A"
  • then commit with "git commit -m "comment about change made" "
  • may be asked for author identity... Use

    git config --global user.email "you@example.com"

    or

    git config --global user.name "Your Name"
  • git pull origin main (ensure main is up to date)
  • git push origin feature/starter-code (pushes commit to github)
